# Gestion Finalistes

Une plateforme simple et pratique pour gérer les finalistes et les élèves.

## Démarrage rapide

### Ouvrir le projet en local (sur ton ordinateur)

C'est super simple ! Deux options :

**Option 1 : Double-clic (la plus facile)**
1. Télécharge tous les fichiers du projet
2. Double-clique sur le fichier `Gestion_Finalistes.html`
3. Ça s'ouvre automatiquement dans ton navigateur ! 

**Option 2 : Avec un serveur local** (si tu as Python ou Node.js)

Si tu as **Python** :
```bash
# Va dans le dossier du projet
cd Gestion-Finalistes

# Lance un serveur
python -m http.server 8000
```

Puis ouvre : `http://localhost:8000/Gestion_Finalistes.html`

---

## Que peux-tu faire avec cette plateforme ?

- **Enregistrer des finalistes** - Ajouter et gérer les données des candidats finalistes
- **Enregistrer des élèves** - Gérer les informations des élèves
- **Consulter les données** - Voir tous les finalistes et élèves enregistrés
- **Vérifier les doublons** - Checker s'il y a des données en double
