# Gestion Finalistes

Bienvenue dans le projet Gestion Finalistes ! 

## Instructions pour ouvrir le projet sur le web
1. Cloner le dépôt en utilisant la commande suivante :
   ```bash
   git clone https://github.com/GermsM/Gestion-Finalistes.git
   ```
2. Accédez au dossier du projet :
   ```bash
   cd Gestion-Finalistes
   ```
3. Lancez le projet en utilisant le serveur local (suivez les instructions spécifiques à votre framework si nécessaire).
4. Ouvrez votre navigateur et allez à l'adresse suivante : `http://localhost:3000` (ou le port que vous avez configuré).

Profitez de votre utilisation !
